# back-node


## Planifiacion de EndPoints

Tareas:
    - Listado: GET /tasks
    - Obtener una tarea: GET /tasks/{id}
    - Crear: POST /tasks
    - Editar: PUT /tasks/{id}
    - Eliminar: DELETE /tasks/{id}
    - Marcar completado: PUT /tasks/{id}?state=completed
    - Marcar pendiente: PUT /tasks/{id}?state=pending
